# Bachelor thesis – Strength verification scripts

Supplementary code for MAST2910 Bachelor's thesis at NTNU 
(Institutt for maskinteknikk og produksjon), spring 2026.

## Contents
- `robot_arm_strength.ipynb` – Cantilever bending verification 
  for the carbon fibre links of the 5-DOF manipulator arm. 
  Implements the analytical model described in Section 7.2.3 
  of the thesis.

## Use of AI tools
The script was developed with implementation assistance from 
Claude Code (Anthropic). All mechanical models, assumptions, 
load cases, safety factors and material parameters were defined 
by the author. Results were cross-checked against analytical 
hand calculations.
